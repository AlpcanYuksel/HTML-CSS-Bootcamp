From etiketleri tek başına anlamsızdır. Bunu çoğu framework'de kullanırız***. action niteliği, form bilgilerinin nereye gönderileceğini belirleriz. method niteliği, post mu get olarak mı gönderilmeli diye belirtiriz.form içerisinde file tipinde input var ise enctype="multipart/form-data" değerini girmemiz gerekiyor

Input çeşitli tipler alır. text, normal yazı tipidir, password şifreler için kullanılır.File, dosya seçmek için kullanılır. checked otomatik olarak işaretli gelir. reset type'ı formu temizler. readonly kullanıcı tarafından değiştirilemez, sadece okunabilir. Bu değer input olarak çekebilmemize engel değildir. required, boş geçilemez alan demek. maxlength, maksimum karakter sayısını belirtir. autocomplete="off", daha önce yazılan değerleri tekrar göstermez. Eğer input etiketine name niteliği belirtilmezse, database'de yakalanamaz. field set, inputun etrafını sarar. legend, başlık belirtir. disabled kullanılamaz yapar.
